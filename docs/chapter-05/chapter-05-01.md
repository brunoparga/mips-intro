CHAPTER 5  —  Representing Characters with Bit Patterns

## Content

**Patterns 
of bits
represent many types of things.**
This chapter shows how
bit patterns are used to represent characters.

**Chapter Topics:**

- ASCII
- Control characters
- Teletype Machines
- `.asciiz` and **null terminated strings**
- Disk files
- Text files
- Binary files
- Executable files
