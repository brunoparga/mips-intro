Shifting Left One Bit

## Answer

An infinite number of digits (or bits).

## Content

Let
us return to paper-and-pencil positional notation.
Here is a binary expression:

```
0001.0100 = 1.2510

```

Here is the same pattern, shifted left by one bit:

```
0010.1000 = 2.5010

```
